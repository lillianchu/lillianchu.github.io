Parental Controls


Skills: Wireframing, Figma, prototyping, user flows, mockups, visual design, design for accessibility


User:

This interface was designed for a mother with an 8-year old child who often uses her phone to play games. Upon turning on the phone, there are three options: unlock the phone to use it normally, go to parental control settings, or set it into restricted mode if handing the phone off to the child.

Card Sorting

When presented with a list of index cards related to parental controls, the 3 main categories that formed were:

  • Restrict Content
  • Limit Usage Time
  • Glorified Spying (which I turned into "Tracking Data")

Using these categories, I wireframed a control panel where the parent user can control several things such as:

  • restricting access to certain apps (ex: restricted use of banking app)
  • providing a time limit for certain apps (ex: Youtube maximum time: 4 hours daily)
  • tracking data (ex: password for Youtube account, what apps that have been used in "Child Mode")
